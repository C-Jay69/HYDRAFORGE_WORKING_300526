# VERTICAL 1: SOFTWARE / SAAS / TECHNOLOGY

## Trigger Keywords
```
TRIGGER IF company name OR business description contains:
["Software", "SaaS", "Tech", "Digital", "AI", "ML", "Cloud",
"Data", "Platform", "App", "Systems", "Solutions", "Cyber",
"Network", "Internet", "Mobile", "API", "Code", "Dev",
"Analytics", "Automation", "Algorithm", "Computing"]
```

## Mandatory Checklist: 25 Items

### IP & Technology (Most Critical for This Vertical)
```
TECH-IP-01: IP OWNERSHIP CHAIN
□ Does contract include rep that ALL IP is owned (not licensed) by Seller?
□ Is there a rep covering contractor/employee IP assignment agreements?
□ Missing → FLAG CRITICAL
Rationale: In software companies, developers often retain IP rights
if assignment agreements were never signed. Buyer could purchase
a product they don't legally own.

TECH-IP-02: OPEN SOURCE COMPLIANCE
□ Is there a rep confirming no "copyleft" open source contamination?
□ Copyleft licenses (GPL, AGPL) can force Buyer to make proprietary
  code public if contaminated software is distributed
□ Missing → FLAG CRITICAL
Rationale: A single contaminated open source component can
legally destroy the entire proprietary codebase's value.

TECH-IP-03: SOURCE CODE ESCROW
□ Is there a source code escrow arrangement for SaaS products?
□ Missing → FLAG HIGH

TECH-IP-04: PATENT ENCUMBRANCES
□ Are there reps covering freedom-to-operate (no third-party patents)?
□ Any pending patent litigation or cease-and-desist letters?
□ Missing → FLAG HIGH

TECH-IP-05: TRADEMARK/BRAND OWNERSHIP
□ Are trademarks registered and owned free of disputes?
□ Missing → FLAG MEDIUM
```

### Data Privacy & Cybersecurity
```
TECH-DATA-01: GDPR/CCPA COMPLIANCE REP
□ Is there a rep confirming compliance with applicable data privacy laws?
□ GDPR (EU users), CCPA (California users), PIPEDA (Canadian users)
□ Missing → FLAG CRITICAL
Rationale: Post-closing GDPR fines can reach 4% of global annual
revenue. Buyer inherits ALL pre-closing violations.

TECH-DATA-02: DATA BREACH HISTORY
□ Rep confirming no unreported data breaches in last 3-5 years?
□ Notification obligations under state breach laws (all 50 states have them)?
□ Missing → FLAG CRITICAL

TECH-DATA-03: CYBERSECURITY POSTURE
□ Rep confirming compliance with industry security standards
  (SOC 2, ISO 27001, NIST)?
□ Missing → FLAG HIGH

TECH-DATA-04: CUSTOMER DATA TRANSFERABILITY
□ Do customer contracts permit transfer of data to Buyer?
□ Do privacy policies permit such transfer?
□ Missing → FLAG CRITICAL
Rationale: If customer data cannot legally transfer, the core
asset of a SaaS business may be non-transferable.

TECH-DATA-05: THIRD PARTY DATA LICENSES
□ Is Seller using third-party data sets under license?
□ Do those licenses transfer to Buyer?
□ Missing → FLAG HIGH
```

### SaaS-Specific Commercial Issues
```
TECH-SAAS-01: RECURRING REVENUE QUALITY
□ Are MRR/ARR figures verified by rep?
□ Churn rate disclosed?
□ Missing → FLAG CRITICAL
Rationale: SaaS valuations are multiple-of-ARR based.
Inflated ARR = inflated purchase price.

TECH-SAAS-02: CUSTOMER CONTRACT ASSIGNABILITY
□ Do SaaS subscription agreements permit assignment to Buyer?
□ Do they contain change-of-control termination rights?
□ Missing → FLAG CRITICAL
Rationale: If top customers have change-of-control termination
rights and exercise them post-closing, ARR collapses.

TECH-SAAS-03: HOSTING/INFRASTRUCTURE DEPENDENCIES
□ Are cloud hosting agreements (AWS, Azure, GCP) assignable?
□ Are there volume commitment penalties on transfer?
□ Missing → FLAG HIGH

TECH-SAAS-04: SOFTWARE LICENSE AGREEMENTS
□ Are all third-party software licenses transferable?
□ Any per-seat licenses that will need repricing post-acquisition?
□ Missing → FLAG HIGH

TECH-SAAS-05: UPTIME/SLA OBLIGATIONS
□ Are there SLA commitments that survive to Buyer?
□ What are the financial penalties for SLA breaches?
□ Missing → FLAG MEDIUM
```

### Employment & Key Person Risk
```
TECH-EMP-01: KEY DEVELOPER RETENTION
□ Are key engineers/developers on retention agreements?
□ Missing → FLAG CRITICAL
Rationale: In software, the product IS the people.
Loss of key developers post-closing can destroy product value.

TECH-EMP-02: NON-SOLICITATION OF EMPLOYEES
□ Does non-compete cover employee solicitation?
□ Missing → FLAG HIGH

TECH-EMP-03: VISA/IMMIGRATION STATUS
□ Are any key employees on H-1B or other work visas?
□ Do those visas require re-sponsorship post-acquisition?
□ Missing → FLAG HIGH
```

### Financial Metrics Verification
```
TECH-FIN-01: REVENUE RECOGNITION POLICY
□ Is revenue recognized at contract signing or delivery?
□ Deferred revenue treatment in working capital calculation?
□ Missing → FLAG HIGH

TECH-FIN-02: CUSTOMER CONCENTRATION
□ What % of revenue comes from top 5 customers?
□ If >30% from single customer → FLAG CRITICAL

TECH-FIN-03: CAPITALIZED SOFTWARE COSTS
□ How much R&D is capitalized vs. expensed?
□ This directly impacts EBITDA and valuation multiples
□ Missing → FLAG HIGH
```

# VERTICAL 2: MANUFACTURING / INDUSTRIAL / AEROSPACE

## Trigger Keywords
```
TRIGGER IF company name OR business description contains:
["Manufacturing", "Industrial", "Aerospace", "Automotive",
"Defense", "Fabrication", "Assembly", "Production", "Plant",
"Factory", "Equipment", "Machinery", "Components", "Parts",
"Engineering", "Precision", "Metal", "Chemical", "Processing"]
```

## Mandatory Checklist: 25 Items

### Environmental (Most Critical for This Vertical)
```
MFG-ENV-01: PHASE I/II ENVIRONMENTAL ASSESSMENT
□ Has a Phase I Environmental Site Assessment been completed?
□ If Phase I flagged issues, has Phase II been completed?
□ Missing → FLAG CRITICAL
Rationale: Environmental cleanup costs can exceed deal value
entirely. Superfund liability has no cap.

MFG-ENV-02: HAZARDOUS MATERIALS INVENTORY
□ Complete inventory of hazardous materials on-site?
□ Historical disposal records available?
□ Missing → FLAG CRITICAL

MFG-ENV-03: ENVIRONMENTAL PERMITS
□ Are all environmental permits listed and transferable?
□ Air emissions permits, water discharge permits, waste permits?
□ Missing → FLAG CRITICAL

MFG-ENV-04: PRE-CLOSING ENVIRONMENTAL LIABILITY ALLOCATION
□ Contract clearly allocates pre-closing environmental liability to Seller?
□ Is allocation limited only to "identified" contamination?
  (If yes → FLAG CRITICAL — unknown contamination falls on Buyer)
□ Missing clear allocation → FLAG CRITICAL

MFG-ENV-05: ENVIRONMENTAL INDEMNITY SURVIVAL
□ Does environmental indemnity survive beyond standard rep survival period?
□ Should survive until applicable statute of limitations
□ Missing → FLAG HIGH
```

### Regulatory & Compliance
```
MFG-REG-01: GOVERNMENT CONTRACTS
□ Is Seller a government contractor? (FAR/DFARS compliance required)
□ Are government contracts assignable? (Often require agency consent)
□ Missing → FLAG CRITICAL

MFG-REG-02: AEROSPACE/DEFENSE CERTIFICATIONS
□ FAA certifications (Part 145, Part 21)?
□ AS9100/ISO 9001 certifications?
□ ITAR/EAR export control compliance?
□ Missing → FLAG CRITICAL
Rationale: Certifications may not transfer automatically.
Loss of certification = loss of ability to operate.

MFG-REG-03: OSHA COMPLIANCE HISTORY
□ OSHA violation history for last 5 years?
□ Pending OSHA investigations?
□ Missing → FLAG HIGH

MFG-REG-04: PRODUCT LIABILITY HISTORY
□ Any product liability claims or recalls?
□ Adequate product liability insurance with tail coverage?
□ Missing → FLAG HIGH
```

### Physical Assets
```
MFG-ASSET-01: EQUIPMENT CONDITION & AGE
□ Independent equipment appraisal completed?
□ Deferred maintenance obligations quantified?
□ Missing → FLAG HIGH

MFG-ASSET-02: EQUIPMENT LIENS
□ UCC lien search completed on all major equipment?
□ Equipment leases vs. owned equipment clearly identified?
□ Missing → FLAG HIGH

MFG-ASSET-03: REAL PROPERTY
□ Environmental condition of owned real property verified?
□ Lease assignment rights confirmed?
□ Missing → FLAG HIGH

MFG-ASSET-04: CAPEX REQUIREMENTS
□ Near-term capital expenditure requirements disclosed?
□ Any equipment at end of useful life?
□ Missing → FLAG MEDIUM
```

### Supply Chain & Customers
```
MFG-SUP-01: SOLE SOURCE SUPPLIER RISK
□ Any single-source suppliers for critical components?
□ Supplier concentration risk quantified?
□ Missing → FLAG HIGH

MFG-SUP-02: CUSTOMER QUALIFICATION REQUIREMENTS
□ Do aerospace/automotive customers require re-qualification
  of supplier after ownership change?
□ Missing → FLAG CRITICAL
Rationale: Re-qualification can take 6-18 months during which
Buyer cannot ship to that customer.

MFG-SUP-03: LONG-TERM SUPPLY AGREEMENTS
□ Are fixed-price long-term contracts in place?
□ Do they have material/energy cost escalation clauses?
□ Missing → FLAG MEDIUM
```

### Labor & Union Issues
```
MFG-LAB-01: UNION/CBA STATUS
□ Is workforce unionized?
□ Is collective bargaining agreement assignable?
□ When does CBA expire?
□ Missing → FLAG CRITICAL

MFG-LAB-02: PENSION/DEFINED BENEFIT OBLIGATIONS
□ Are there defined benefit pension plans?
□ What is the funding status/underfunding amount?
□ Missing → FLAG CRITICAL
Rationale: Pension underfunding is a direct liability that
transfers to Buyer and can be enormous.

MFG-LAB-03: WARN ACT COMPLIANCE
□ If plant closure or mass layoff planned post-closing,
  60-day WARN Act notice required
□ Missing → FLAG HIGH
```

# VERTICAL 3: HEALTHCARE / LIFE SCIENCES / PHARMA

## Trigger Keywords
```
TRIGGER IF company name OR business description contains:
["Health", "Medical", "Pharma", "Biotech", "Clinical",
"Hospital", "Therapy", "Drug", "Device", "Diagnostic",
"Lab", "Patient", "Care", "Surgical", "Dental", "Vision",
"Mental Health", "Behavioral", "Life Sciences", "Genomic"]
```

## Mandatory Checklist: 25 Items

### Regulatory (Most Critical for This Vertical)
```
HEALTH-REG-01: FDA COMPLIANCE
□ FDA registration and facility inspection history?
□ Any Warning Letters, 483 Observations, or consent decrees?
□ Missing → FLAG CRITICAL

HEALTH-REG-02: DEA REGISTRATION
□ If controlled substances involved, DEA registration status?
□ DEA registrations are non-transferable — new registration required
□ Missing → FLAG CRITICAL

HEALTH-REG-03: STATE HEALTHCARE LICENSES
□ All state-specific healthcare licenses identified?
□ License transferability confirmed?
□ Missing → FLAG CRITICAL
Rationale: Healthcare licenses are highly jurisdiction-specific
and often non-transferable. Loss = inability to operate.

HEALTH-REG-04: MEDICARE/MEDICAID ENROLLMENT
□ CMS enrollment status verified?
□ Any OIG exclusions or CMS billing suspensions?
□ Missing → FLAG CRITICAL
Rationale: OIG exclusion = cannot bill federal healthcare programs
= instant revenue destruction.

HEALTH-REG-05: CERTIFICATE OF NEED
□ If applicable (hospital, imaging center), CON status verified?
□ CON transfer requirements confirmed?
□ Missing → FLAG HIGH
```

### HIPAA & Patient Data
```
HEALTH-HIPAA-01: HIPAA COMPLIANCE PROGRAM
□ Documented HIPAA compliance program in place?
□ BAA (Business Associate Agreements) with all vendors?
□ Missing → FLAG CRITICAL

HEALTH-HIPAA-02: PHI BREACH HISTORY
□ Any unreported PHI breaches in last 6 years?
□ HHS Office for Civil Rights investigations?
□ Missing → FLAG CRITICAL
Rationale: HIPAA penalties up to $1.9M per violation category
per year. Buyer inherits ALL pre-closing violations.

HEALTH-HIPAA-03: PATIENT DATA TRANSFERABILITY
□ Can patient records legally transfer to Buyer?
□ Patient notification requirements on ownership change?
□ Missing → FLAG CRITICAL
```

### Fraud & Abuse
```
HEALTH-FRAUD-01: STARK LAW COMPLIANCE
□ Physician self-referral arrangements reviewed for Stark Law compliance?
□ Missing → FLAG CRITICAL
Rationale: Stark Law violations = False Claims Act liability
= treble damages + exclusion from Medicare.

HEALTH-FRAUD-02: ANTI-KICKBACK STATUTE
□ Any financial relationships with referral sources reviewed?
□ Missing → FLAG CRITICAL

HEALTH-FRAUD-03: FALSE CLAIMS ACT EXPOSURE
□ Any qui tam (whistleblower) actions pending or threatened?
□ Missing → FLAG CRITICAL

HEALTH-FRAUD-04: GOVERNMENT INVESTIGATION HISTORY
□ Any DOJ, HHS-OIG, or state AG investigations in last 5 years?
□ Missing → FLAG CRITICAL
```

### Clinical & Product-Specific
```
HEALTH-CLIN-01: CLINICAL TRIAL AGREEMENTS
□ Are ongoing clinical trials assignable?
□ IRB approval status for trials?
□ Missing → FLAG HIGH

HEALTH-CLIN-02: DRUG/DEVICE APPROVALS
□ FDA 510(k) clearances or PMA approvals transferable?
□ NDA/ANDA ownership clearly assigned?
□ Missing → FLAG CRITICAL

HEALTH-CLIN-03: PRODUCT LIABILITY/RECALL HISTORY
□ Any product recalls, adverse event reports (MDRs)?
□ Tail coverage for pre-closing product liability?
□ Missing → FLAG HIGH

HEALTH-CLIN-04: REIMBURSEMENT RISK
□ Are products/services dependent on specific CPT codes?
□ Any pending reimbursement rate changes?
□ Missing → FLAG HIGH
```

```

## Mandatory Checklist: 25 Items

### Regulatory Licenses (Most Critical for This Vertical)
```
FIN-LIC-01: REGULATORY LICENSE INVENTORY
□ Complete list of ALL regulatory licenses obtained?
  - Banking: OCC, Federal Reserve, FDIC, state banking licenses
  - Securities: FINRA, SEC registrations (BD, IA, Transfer Agent)
  - Insurance: State insurance licenses (all 50 states potentially)
  - Payments: Money transmitter licenses (state by state)
□ Missing → FLAG CRITICAL
Rationale: Financial services licenses are often
non-transferable and deal can be blocked entirely.

FIN-LIC-02: CHANGE OF CONTROL APPROVALS
□ Which licenses require regulatory approval for ownership change?
□ Timeline for regulatory approval obtained?
□ Missing → FLAG CRITICAL
Rationale: Banking M&A approvals can take 12-18 months.
Deal timeline must account for this.

FIN-LIC-03: REGULATORY EXAMINATION HISTORY
□ Last 3 examination reports reviewed?
□ Any MRAs (Matters Requiring Attention) outstanding?
□ Missing → FLAG CRITICAL

FIN-LIC-04: ENFORCEMENT ACTIONS
□ Any consent orders, cease and desist orders, or MOUs?
□ Any pending regulatory investigations?
□ Missing → FLAG CRITICAL
```

### Capital & Financial Condition
```
FIN-CAP-01: REGULATORY CAPITAL ADEQUACY
□ Is entity meeting minimum capital requirements?
□ What is the capital impact of the transaction itself?
□ Missing → FLAG CRITICAL

FIN-CAP-02: LOAN PORTFOLIO QUALITY (Banking)
□ NPL (Non-Performing Loan) ratio?
□ Loan loss reserve adequacy?
□ Any classified or criticized assets?
□ Missing → FLAG CRITICAL

FIN-CAP-03: RESERVE ADEQUACY (Insurance)
□ Actuarial certification of loss reserves?
□ Any reserve strengthening in last 3 years?
□ Missing → FLAG CRITICAL

FIN-CAP-04: LIQUIDITY POSITION
□ Liquidity coverage ratio (if applicable)?
□ Any liquidity facilities or contingent funding arrangements?
□ Missing → FLAG HIGH
```

### Compliance & AML
```
FIN-AML-01: BSA/AML COMPLIANCE PROGRAM
□ Documented BSA/AML compliance program?
□ SAR filing history and FinCEN examinations?
□ Missing → FLAG CRITICAL
Rationale: AML failures = criminal liability for Buyer
post-acquisition. DOJ has prosecuted acquirers.

FIN-AML-02: SANCTIONS COMPLIANCE
□ OFAC compliance program in place?
□ Any OFAC violations or SDN list customers?
□ Missing → FLAG CRITICAL

FIN-AML-03: CRA COMPLIANCE (Banking)
□ Community Reinvestment Act rating?
□ Poor CRA rating can block acquisition approval
□ Missing → FLAG HIGH

FIN-AML-04: CONSUMER PROTECTION
□ CFPB examination history?
□ Any UDAAP violations or complaints?
□ Missing → FLAG HIGH
```

### Portfolio/Book of Business
```
